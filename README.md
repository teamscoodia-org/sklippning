# S Klipning (Next.js)

This project is a Next.js App Router site built with TypeScript, shadcn/ui, and Tailwind CSS.

## Run locally

```sh
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` - start Next.js dev server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - ESLint
- `npm test` - Vitest
